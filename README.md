# Snake Upgrade (Pygame)

Python과 Pygame으로 만든 업그레이드 버전 스네이크 게임입니다.

일반 스네이크 규칙에 더해, 먹이를 먹으면 뱀의 방향이 뒤집히는 특수 규칙과
장애물, 특수 먹이, 제한 시간, 최고 점수 저장 기능이 들어 있습니다.

## 주요 기능

- 방향키 또는 `WASD`로 이동
- 일반 먹이를 먹으면 점수 증가
- 일반 먹이를 먹을 때마다 뱀이 뒤집힘
- 30초 안에 먹이를 먹지 못하면 게임 오버
- 레벨이 오를수록 속도와 장애물 증가
- 금색 특수 먹이: 보너스 점수 `+50`
- 파란색 특수 먹이: 속도 감소 효과
- 콤보 점수 시스템
- 최고 점수 자동 저장
- 시작 화면, 일시정지 화면, 게임 오버 화면
- 영어/한국어 UI 전환 지원

## 조작법

| 키 | 기능 |
| --- | --- |
| 방향키 / `WASD` | 뱀 이동 |
| `SPACE` | 시작 / 일시정지 / 계속 |
| `K` | 영어 / 한국어 전환 |
| `R` | 재시작 |
| `Q` 또는 `ESC` | 종료 |

## 실행 방법

```bash
pip install pygame
python main.py
```

## 게임 규칙

1. 뱀을 움직여 빨간 먹이를 먹습니다.
2. 먹이를 먹으면 점수가 오르고 뱀이 길어집니다.
3. 먹이를 먹은 순간 뱀의 머리와 꼬리가 뒤집힙니다.
4. 벽, 자기 몸, 장애물에 부딪히면 게임 오버입니다.
5. 먹이를 먹은 뒤 30초 안에 다음 먹이를 먹어야 합니다.
6. 레벨이 올라갈수록 속도가 빨라지고 장애물이 늘어납니다.
7. 보드를 모두 채우면 승리합니다.

## 저장 파일

게임을 실행하면 최고 점수를 저장하기 위해 `high_score.json` 파일이 자동으로 생성될 수 있습니다.

이 파일은 게임 기록용 파일이며, 삭제하면 최고 점수가 초기화됩니다.

## 필요 환경

- Python 3
- Pygame
